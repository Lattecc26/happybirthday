# 生日快乐祝福页面 🎂

一个精美的在线生日祝福页面，包含动态效果和个性化祝福功能。

## 功能特色 ✨

- 🎨 **精美设计**：磨砂玻璃效果登录框，动态背景
- 🌟 **动态效果**：流星雨、星星闪烁、礼花爆炸
- 🎵 **背景音乐**：自动播放生日音乐
- 📝 **个性化祝福**：输入名字生成专属祝福
- 📱 **响应式设计**：适配手机和电脑端
- 🎆 **庆祝效果**：彩带飘落、礼花绽放

## 快速开始 🚀

### 在线访问
通过 GitHub Pages 访问：
https://lattecc26.github.io/happybirthday/happybirthday.html

### 本地运行
1. 下载所有文件到本地
2. 用浏览器打开 `happybirthday.html`
3. 输入名字，点击"进入"开始体验

## 文件结构 📁

```
happybirthday/
├── happybirthday.html    # 主页面文件
├── cake.png              # 生日蛋糕图片
├── music.ogg             # 背景音乐文件
└── README.md             # 项目说明
```

## 使用说明 📖

1. **登录页面**：输入您的名字
2. **主页面**：观看流星雨和个性化祝福
3. **蛋糕页面**：点击"再点一下"进入蛋糕祝福页面
4. **庆祝效果**：享受彩带、礼花等庆祝动画
